# TPE Arquitectura de las Computadoras (Computers Architecture Assignment) #
Computers Architecture Assignment
Pure64 based 64bit OS with the following features:
- Mouse Driver
- Video Driver
- Keyboard Driver
- Modules loading
- Paging
